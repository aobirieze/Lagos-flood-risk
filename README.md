# Lagos Flood Risk Analysis

## Project Overview
Riverine and coastal flood risk mapping for Lagos, Nigeria, 
built as part of a catastrophe risk modelling portfolio project.

## Key Findings
- Lagos faces significantly higher riverine flood risk than 
  coastal flood risk
- Key high-risk areas: Lagos Island, Apapa port, and Ikorodu corridor
- Coastal flood exposure is concentrated near Eko Atlantic — 
  one of Lagos's most expensive new developments
- Risk is driven by drainage infrastructure failure and rapid 
  urbanisation rather than sea level exposure alone

## Data Sources
- WRI Aqueduct Floods (1-in-100 year return period)
- OpenStreetMap basemap via QGIS QuickMapServices
